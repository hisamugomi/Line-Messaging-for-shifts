Setup Instructions
  1. LINE API Setup: Set your LINE Channel Access Token as an environment variable: LINE_CHANNEL_ACCESS_TOKEN
  2. Excel Format: Ensure your Excel file contains the required columns with proper data types.
  3. LINE User IDs: Make sure all LINE user IDs are valid and the bot has permission to send messages.
  4. Testing: Start with a small test file to verify everything works correctly.
