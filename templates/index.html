LINE シフト自動化

Excelスケジュールをアップロードし、データをプレビューして、LINE経由でシフト通知を自動送信します。

Excelファイルのアップロード
Excelファイルには以下の列が含まれている必要があります： employee_name, line_user_id, shift_date, start_time, end_time
データプレビュー
従業員名 LINE ID シフト日 時間
メッセージを送信する前に、データを注意深く確認してください。
セットアップ手順
  1. LINE API セットアップ: LINE Channel Access Token を環境変数として設定してください： LINE_CHANNEL_ACCESS_TOKEN
  2. Excel 形式: Excelファイルに必要な列が適切なデータ型で含まれていることを確認してください。
  3. LINE ユーザーID: すべてのLINEユーザーIDが有効で、ボットがメッセージを送信する権限を持っていることを確認してください。
  4. テスト: 小さなテストファイルから始めて、すべてが正しく動作することを確認してください。
